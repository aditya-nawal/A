Malicious Payload The program generates a .c file with the following payload: #include int main () { printf ( "You are stupid!\n" ); return 0 ; } How It Works Scans all files in the current directory. Skips . and .. entries. Generates a .c file for each filename, appending the .c extension. Inserts the malicious payload into the newly created file. Compiles the .c file into an executable using GCC: On Windows: Creates a .exe file. On Linux/Unix: Creates a binary file without an extension. Deletes the .c file after compilation. Usage Prerequisites GCC installed on your system. A directory containing files to process. Steps Clone the repository: git clone < repository_url > cd < repository_name > Compile the program: gcc -o maarsh maarsh.c Run the executable: ./maarsh Malicious executables will be created for each file in the current directory.
